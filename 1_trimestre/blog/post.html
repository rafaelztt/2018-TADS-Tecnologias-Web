Formulários

Postado por Rafael Zottesso em 06 de março de 2018.

O HTML "form" elemento define um formulário que é usado para coletar a entrada do usuário.

Um formulário HTML contém elementos de formulário. Elementos de formulário são diferentes tipos de elementos de entrada, como campos de texto, caixas de seleção, botões de rádio, botões de envio, e muito mais.

O "input" elemento é o mais importante elemento de formulário. Ele pode ser exibido de várias maneiras, dependendo do tipo de atributo.

Algumas exemplos são:

Tipo Descrição
text Define um campo de entrada de texto de uma linha
radio Define um botão de rádio (para a seleção de uma das muitas escolhas)
submit Define um botão de envio (para enviar o formulário)
